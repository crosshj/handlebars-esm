# @fiug/handlebars-esm
a port of Handlebars to esm compatable module


## Usage

from npm

```
npm i @fiug/handlebars-esm --save
```

```javascript
import Handlebars from "@fiug/handlebars-esm";
```

or directly from unpkg

```javascript
import Handlebars from "https://unpkg.com/@fiug/handlebars-esm";
```
